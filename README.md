# ma-garin.github.io

Yuki Fujimagari / ma-garin のQAエンジニア向けポートフォリオサイトです。

公開URL:

https://ma-garin.github.io/

## 構成

- `index.html`: トップページ
- `about.html`: プロフィール
- `capabilities.html`: 強み・専門性
- `experience.html`: 職務経験・登壇実績
- `works.html`: 公開実績
- `principles.html`: 品質保証における考え方
- `contact.html`: 公開プロフィールリンク
- `404.html`: 存在しないURLにアクセスしたときのページ
- `styles.css`: 共通スタイル
- `.nojekyll`: GitHub PagesでJekyll処理を無効化するためのファイル

## 公開方針

- 本人が公開している範囲の実名、職務経歴、登壇実績を掲載する
- 住所、電話番号、個人メールアドレスは掲載しない
- 公開可能な制作物、学習記録、検証観点、公開プロフィールへのリンクを中心に掲載する
- 実務案件や顧客情報は、公開可能な範囲に抽象化する

## 更新方法

1. 対象ページまたは `styles.css` を編集する
2. ローカルで表示を確認する
3. 変更をコミットして `main` に反映する

このリポジトリは GitHub Pages のユーザーサイト用リポジトリです。
